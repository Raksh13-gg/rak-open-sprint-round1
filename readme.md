AI Brain Cell Detector

  An interactive AI-based web application that analyzes a user's facial expression using the webcam and humorously estimates the number of "brain cells" based on detected emotions. The project uses face detection and emotion recognition to generate a fun meme-style result.

  Features:-

-> Real-time Face Detection using webcam

-> Emotion Recognition (happy, sad, surprised, neutral, etc.)

-> Brain Cell Estimation based on detected emotion

->Automatic Meme Generation depending on emotion

-> Reset Button to scan again

-> Scanning Sound Effect for better user interaction

Technologies Used:-

-> HTML5 – Structure of the application

-> CSS3 – Styling and layout

-> JavaScript – Application logic

-> Face API.js – Face detection and emotion recognition

-> Webcam API – Accessing the user's camera



How It Works:-

1. The application accesses the user's webcam.

2. Face detection identifies the user's face in real time.

3. Emotion detection analyzes facial expressions.

4. Based on the detected emotion, the system:

5. assigns a fun brain cell count

6. displays a related meme.

7. Users can reset the scan and try again.

How to run the project:-

First, download or clone the project repository to your system. Open the project folder and run the main HTML file in a web browser. When the application starts, allow camera permission so the system can access the webcam. Once the camera is active, the application will detect the face, analyze the emotion, and display the brain cell estimation along with a meme.

Note:
This project is developed for learning and demonstration purposes. The brain cell estimation shown by the application is generated based on detected facial emotions and is intended only for entertainment. It does not represent any real or scientific measurement.
